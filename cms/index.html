COMPLAINT

About

In Order to attend the student's complaints, a grievance redressal cell has been set up.

  • User registration

    The student's accounts into the system.

    Input: Name, Reg.no, Email, password, phone.

    Output: Student is registered to system.

  • USER LOGIN

    Only authorized person can enter into the system.

    • Input: username/reg.no, password.

    • Output: student is logged in.

  • COMPLAINT REGISTRATION

    Students to register their grievances.

    • Input: Department, complaint

    • Output: Complaint is registered.

  • REPORT / SOLUTION

    Generates complaint report by both the admin and Grievance management cell.

    • Input: start date, end date, status
